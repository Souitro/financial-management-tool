# Financial Management Tool

A web-based application built to help users manage their finances, track expenses, and plan budgets.

## Features
- Track daily income and expenses
- Generate financial reports
- Secure user authentication

## Tech Stack
- **Backend:** PHP / XAMPP
- **Database:** MySQL
- **Frontend:** HTML, CSS, JavaScript

## How to Run Locally
1. Clone this repository.
2. Move the project folder into your `xampp/htdocs/` directory.
3. Start Apache and MySQL in the XAMPP Control Panel.
4. Import the database (if applicable).
5. Open `http://localhost/1/1/Financial Management Tool` in your browser.
